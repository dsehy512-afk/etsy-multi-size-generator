# Etsy Multi Size Generator

## Overview
This project converts a single image into multiple high-resolution (300 DPI) print sizes required for Etsy digital downloads.

Instead of manually resizing files across tools like Canva or Photoshop, this system automates the process into a single step.

## Problem
Etsy sellers currently:
- Manually resize images multiple times
- Export each size individually
- Package files into ZIP folders

This process can take 1–2 hours per listing.

## Solution
Upload one image → Select desired sizes → Generate all outputs instantly.

Supported sizes include:
- 24x36
- 18x24
- 16x20
- 11x14
- 8x10

## Goal
Build a simple tool (and future app) that:
- Saves time
- Standardizes output quality (300 DPI)
- Eliminates repetitive manual work

## Future Vision
- Drag-and-drop interface
- Batch processing
- Etsy-ready ZIP packaging
- API integration for marketplaces
